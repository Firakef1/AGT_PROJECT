# GubaeTech

A centralized digital system built for ASTU Gibi Gubae (a university church fellowship organization). The system manages multiple subdivisions, each handling their own members, budgets, inventory, and internal activities.

## Features

- Member management across divisions
- Financial tracking and budgeting
- Inventory management
- Activity reporting
- Admin dashboard for church leaders

## Tech Stack

- React 19
- Vite
- Modern web technologies

## Getting Started

```bash
npm install
npm run dev
```

## Development

```bash
npm run build
npm run preview
```
